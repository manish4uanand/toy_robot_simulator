# toy_robot_simulator

=> Execute with following command:
> ruby toy_robo.rb 

## After Asking for command for simulatore use following example:
### Example Input and Output:
```i. 
PLACE 0,0,NORTH
MOVE
REPORT
Output: 0,1,NORTH
```
```
ii. 
PLACE 0,0,NORTH
LEFT
REPORT
Output: 0,0,WEST
```
```
iii. 
PLACE 1,2,EAST
MOVE
MOVE
LEFT
MOVE
REPORT
Output: 3,3,NORTH
```
